**Y7** is an upcoming version of ASCIICKER, which will introduce fighting mechanics.

+ Upcoming features

* Fighting mechanics[((bibcite fightingSource))]
* [[[wiki:NPCs|Non-player Characters]]][((bibcite avaSource))]
* Current loading screen will be replaced by an ASCII loading screen or splash[((bibcite splashSource))]
* Health Points will be displayed on the red bar[((bibcite hpSource))]. The blue bar will not change[((bibcite blueBarSource))]
Y7 will not be multiplayer[((bibcite multiSource))], however this will be re-implemented in later versions[((bibcite multiReimpSource))].
